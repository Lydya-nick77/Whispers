# Whispers

Addon to replace the FFXI chat and combat log interface.

The chat windows has tabs to easily manage incoming messages incoming from LS, Party and Tells.
![Chat window](main.png)

Combat Log configurable with colors for different actions
![Combat Log window](combatlog.png)

Use the command /whispers to access the configuration window.
![Config window](config.png)

List of all the commands:
```
/whispers           Opens the config menu
/whispers chat      Toggle the display of the chat window
/whispers help      Display the list of commands
/whispers combat    Toggle the display of the combat log window
```
